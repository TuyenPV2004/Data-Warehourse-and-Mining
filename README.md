# README - Data_Warehourese

## 1) Thư mục dự án chính
- Thư mục làm việc chính: `Data_Warehourese`
- Gồm:
  - `backend`
  - `frontend`

## 2) Chạy backend
```powershell
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## 3) Chạy frontend
```powershell
cd frontend
npm run dev
```

## 4) URL
- Frontend: `http://localhost:5173`
- Backend docs: `http://localhost:8000/docs`
